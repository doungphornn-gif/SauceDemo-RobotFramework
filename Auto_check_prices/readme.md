### Test Suite: Check Product Prices
ตรวจสอบรายละเอียดการทดสอบความถูกต้องของการแสดงราคาสินค้า (Price Verification) บนหน้า Inventory ของเว็บไซต์ SauceDemo

### จุดประสงค์การทดสอบ (Test Scenario)
เพื่อตรวจสอบว่าระบบสามารถดึงข้อมูลราคาสินค้าทั้ง 6 รายการมาแสดงผลบนหน้าเว็บไซต์ได้อย่างถูกต้อง ตรงตามข้อมูลมาตรฐาน (Expected Price) และไม่มีความผิดเพี้ยนของการแสดงผลเมื่อเข้าใช้งานด้วย User ที่แตกต่างกัน

### รายละเอียด Test Cases
เป็น ID User ที่สามารถเข้าใช้งานได้ไปที่หน้าเลือกซื้อสินค้าได้และราคาสินค้าต้องมีความถูกต้องทั้งหมด
