### Test Suite: Auto Login Users
หน้านี้คือรายละเอียดการทดสอบสั่งสินค้าหลายชิ้นและเช็คยอดตะกร้าของ User ID ปกติที่สามารถใช้งานได้

### จุดประสงค์การทดสอบ (Test Scenario)
สามารถสั่งสินค้าได้ทุกชิ้น และสินค้าในตะกร้าตรงกับสินค้าที่กด add to cart

### สรุปผลการรันบอท (Test Execution Report)
ดูรูปผลลัพธ์การรัน 3 Passed / 2 Failed จากหน้าจอ Terminal ด้านล่าง
